AI 오버뷰 답변 생성Solar 호출 #4
검증된 소스 문서를 바탕으로 Solar가 요약 답변을 생성합니다. 링크를 나열하는 것이 아니라, 사용자의 질문에 직접 답하는 형태입니다. 출처를 명시하여 신뢰성을 확보합니다.

왜 업스테이지여야만 이 구조가 성립하는가
전제 조건
AI 검색은 쿼리당 LLM을 4회 호출해야 함
쿼리 이해 → 하이브리드 검색 → 리랭킹 → 답변 생성. 이 중 어느 하나라도 빠지면 품질이 급락합니다.
문제
다음의 쿼리당 수익은 ~10원 — 외부 API 4회 호출 비용(5~8원)을 감당할 수 없음
네이버(HyperCLOVA X 자체 보유)나 구글(Gemini)과 달리, 다음은 자체 LLM이 없었습니다. 외부 API 의존 시 규모가 커질수록 적자가 커지는 구조.
해법
업스테이지의 Solar 모델을 온프레미스로 운영 → 추론 비용 1/5~1/8 수준
업스테이지가 AXZ의 모회사이기 때문에 (1) 모델 라이선스 비용 없음, (2) AXZ 전용 파인튜닝 가능, (3) 한국어 검색 특화 최적화가 가능합니다. 외부 고객이 아닌 내부 시너지.
결과
다음이 네이버·구글과 동등한 AI 검색을 구현할 수 있는 유일한 경로
자체 LLM을 개발하려면 수천억 + 수년이 필요. 업스테이지 인수를 통해 즉시 활용 가능한 한국어 특화 LLM을 확보한 것이 이 딜의 핵심 시너지.
